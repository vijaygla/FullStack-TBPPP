| Medium              | md           | 768 - 992 px   |       720 px            |
| Large               | lg           | 992 - 1200 px  |       960 px            |
| Extra Large         | xl           | 1200 - 1400 px |       1140 px           |
| Extra Extra Large   | xxl          | > 1400 px      |       1320 px           |
